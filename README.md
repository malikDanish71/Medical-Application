# Medical Healthcare App

## Description
A mobile app that allows patients to view a list of doctors, check their qualifications, experience, and specialization, and book appointments. 

## Features
- Browse doctors with CardView UI
- View doctor details (name, qualification, specialization, experience)
- Make appointments
- User-friendly interface

![Doctor List](app/src/main/res/drawable/screenshot.png)

## Installation
1. Clone the repository
2. Open in Android Studio
3. Build and run on an emulator or device

## Tech Stack
- Android Studio
- Kotlin / Java
- XML layouts
- CardView for UI
